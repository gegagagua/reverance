# reverance

Real estate landing site built with Next.js.

## Getting started

```bash
yarn install
yarn dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `yarn dev` — development server
- `yarn build` — production build
- `yarn lint` — ESLint
- `yarn typecheck` — TypeScript
- `yarn format:check` — Prettier
